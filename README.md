# Mohammed Shamil Portfolio

Dark-mode bilingual (Arabic/English) personal portfolio website built from PRD v1.0.

## Features

- Single-page portfolio layout
- Arabic (RTL) and English (LTR) toggle
- Dark futuristic design with accent gradients
- Responsive navbar with mobile menu
- Sections: Hero, About, Education, Skills, Experience, Contact
- Smooth scrolling and scroll-reveal animations

## Structure

- `index.html`
- `css/style.css`
- `css/rtl.css`
- `js/main.js`
- `js/lang.js`
- `data/content.json`
- `assets/images/profile.jpg` (add your image here)

## Run locally

Open `index.html` directly in browser, or use a static server:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.
